API Documentation

📦 Products Endpoints

GET    /api/products                 → Get all products (Public)
GET    /api/products/:id             → Get product by ID (Public)
POST   /api/products                 → Create new product (Protected)
PUT    /api/products/:id             → Update product (Protected)
DELETE /api/products/:id             → Delete product (Protected)

📋 Orders Endpoints

GET    /api/orders                  → Get all orders (Protected)
GET    /api/orders/:id               → Get order by ID (Protected)
POST   /api/orders                   → Create new order (Public)
PUT    /api/orders/:id               → Update order status (Protected)
DELETE /api/orders/:id               → Delete order (Protected)

🔌 Authentication & Utility Endpoints

POST   /api/login                   → Admin Authentication (Public)
POST   /api/upload                  → Upload Artwork Image (Protected)
GET    /api/health                  → Health check (Public)
GET    /api/stats                    → Database statistics (Protected)
